Назначение макета
Макет разработан для сайта страховой компании, специализирующейся на ОСАГО и КАСКО. Предназначен для демонстрации пользовательского интерфейса и сценариев взаимодействия клиента с сервисом.

Реализованные экраны:

Главная страница
Регистрация (3-шаговая форма)
Авторизация
Личный кабинет (с тремя вкладками)
Информационная страница (о компании)

Пользовательские сценарии:

Ознакомление с услугами компании на главной странице
Создание учётной записи (регистрация в три шага)
Вход в существующий аккаунт (авторизация)
Просмотр и редактирование профиля в личном кабинете
Получение информации о компании
